## Setup

Install dependencies:

```sh
cd $PROJECT_ROOT
npm i
```

## How to Run

### Development

Run the development server:

```sh
npm run dev
```

### Production

Build for production:

```sh
npm run build
```

Then, run production code:

```sh
npm run start
```
